# 📊 Excel Sales Dashboard

## 📌 Project Overview
This project is an interactive Sales Dashboard built using Microsoft Excel. It helps analyze sales performance, profit, orders, and regional trends through dynamic visualizations.

---

## 📈 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Profit Margin

---

## 📊 Dashboard Features

- Sales by Category
- Sales by State
- Monthly Sales Trend
- Top Performing Products
- Regional Performance
- Interactive Slicers

---

## 🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## 💡 Key Insights

- Identified top-performing states.
- Compared category-wise sales.
- Analyzed monthly sales trends.
- Highlighted high-profit products.
